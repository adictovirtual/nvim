.vim
====

To use this configuration
----

If you already have a `.vim` folder, back it up:
```sh
mv ~/.vim ~/.vim.bak
```

Clone the repo:
```sh
git clone --recursive git://github.com/spicydonuts/.vim.git ~/.vim
```

To pull updates later:
----

```sh
cd ~/.vim && git pull && git submodule update --recursive
```
